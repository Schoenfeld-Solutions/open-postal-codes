D-A-CH post code data

Open Postal Codes

Versioned CSV, JSON, and XML files for D-A-CH post codes, distributed through a static GitHub Pages file API with hashes, record counts, and compressed downloads. Records include one location-primary marker per post code plus ranks for city-level post code ordering.
