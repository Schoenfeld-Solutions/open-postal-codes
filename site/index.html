Quickstart

The API is a static file API. No token, account, or server-side negotiation is required.

Base URL 
https://schoenfeld-solutions.github.io/open-postal-codes/api/v1
Fetch the manifest 
curl -fsS https://schoenfeld-solutions.github.io/open-postal-codes/api/v1/index.json
Download Germany CSV 
curl -fsS -O https://schoenfeld-solutions.github.io/open-postal-codes/api/v1/de/post_code.csv
Use compressed data 
curl -fsS https://schoenfeld-solutions.github.io/open-postal-codes/api/v1/ch/post_code.csv.gz | gzip -dc

Schema

CSV uses the field order below. JSON and XML publish the same fields with booleans and ranks typed naturally in JSON.

Field Meaning
code Post code; DE has five digits, AT and CH have four digits.
city City or place label associated with the post code.
country ISO country code: DE, AT, or CH.
state Bundesland for DE and AT; canton for CH.
county Lower administrative area when available.
time_zone Windows time zone identifier; D-A-CH uses W. Europe Standard Time.
is_primary_location True for the highest-ranked row within a country and post code.
location_rank Rank of this place within the same post code.
postal_code_rank Rank of this post code within the same normalized place.
source Either postal_boundary or address_fallback.
evidence_count Accepted address evidence count behind the row.
